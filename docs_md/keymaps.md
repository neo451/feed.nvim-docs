# Keymaps

By default, feed.nvim will not set any keymaps for you, press `?` in to see buffer-local keymaps.

## Index buffer

| action     | key      |
| ---------- | -------- |
| hints      | `?`      |
| dot_repeat | `.`      |
| undo       | `u`      |
| redo       | `<C-r>`  |
| entry      | `<CR>`   |
| split      | `<M-CR>` |
| browser    | `b`      |
| refresh    | `r`      |
| update     | `R`      |
| search     | `s`      |
| yank_url   | `Y`      |
| untag      | `-`      |
| tag        | `+`      |
| quit       | `q`      |

## Entry buffer

| action   | key |
| -------- | --- |
| hints    | `?` |
| browser  | `b` |
| next     | `}` |
| prev     | `{` |
| full     | `f` |
| search   | `s` |
| untag    | `-` |
| tag      | `+` |
| urlview  | `r` |
| yank_url | `Y` |
| quit     | `q` |
